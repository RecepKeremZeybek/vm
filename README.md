# vm

 ### to clone the project
 
 ```cmd
 git clone https://github.com/gm-zeybek/vm.git 
 ```
 
  ### got to the project directory
 
 ```cmd
 cd vm
 ```

 ### to run the project from cmd and install all dependencies

 ```cmd
 mvn clean install 
 ```

 ### to run all tests
 
 ```cmd
 mvn test
 ```
